CORE CAPABILITIES

Framework Architecture

◈

Hyperspectral Imaging

Processing 16-band hyperspectral cubes with radiometric correction using dark/white reference normalization for clinical-grade accuracy.

  • Dark/White Reference Correction
  • Patch-based Classification (32×32)
  • PCA Dimensionality Reduction
◇

Spectral–Spatial Learning

Dual-branch feature extraction combining spectral signatures with GLCM spatial texture features for comprehensive tissue characterization.

  • Mean/Variance Spectral Features
  • GLCM Texture Analysis
  • Spectral Index Computation
⊕

Fusion Classifiers

Softmax probability fusion of Random Forest and SVM predictions for robust multi-class medical image classification with confidence scoring.

  • Random Forest (Spectral Branch)
  • SVM (Spatial Branch)
  • Softmax Probability Fusion
